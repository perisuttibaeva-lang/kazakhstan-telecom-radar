# Телеком-радар Казахстана

Рабочий сайт-мониторинг открытых новостей по телеком-рынку Казахстана.

## Как открыть

1. Запустите файл `start-site.bat`.
2. В браузере откроется сайт: http://localhost:3001
3. Нажмите `Запустить поиск` или `Обновить`.

## Что делает сайт

- ищет новости из открытых Google News RSS-источников;
- отслеживает Beeline, Kcell/Activ, Tele2, Altel, Казахтелеком и общий рынок;
- выделяет тарифы, регуляторику, инфраструктуру и рыночные события;
- оценивает важность новостей;
- хранит архив в `data/archive.json`;
- показывает ленту, фильтры, краткую сводку и аналитику.

## Ручной запуск

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File ".\start-radar.ps1" -Port 3001
```

После запуска откройте:

```text
http://localhost:3001
```

## Важно

Если открыть `index.html` двойным кликом, сайт тоже попробует найти локальный сервер, но надежнее открывать именно `http://localhost:3001`.

Для публичной публикации в интернете потребуется хостинг с серверной частью, например Render, Railway, VPS или аналогичный сервис.
